1. Кнопки

Используются для активации действий на странице. Доступны следующие вариации:

  • Размеры: маленькая, средняя, большая
  • Классы: .button--primary, .button--secondary, .button--disabled

2. Поля ввода

Используются для ввода текста. Доступны следующие состояния:

  • Состояние нормальное
  • Состояние фокуса

3. Карточки

Карточки используются для отображения информации. Доступны следующие вариации:

  • Размеры: маленькая, средняя, большая
  • С изображением или без изображения

Flexbox

Используется для создания гибких, адаптивных макетов, которые можно легко выравнивать и распределять элементы.

  • Гибкость в распределении пространства
  • Выравнивание элементов по вертикали и горизонтали
  • Поддержка адаптивных интерфейсов

Grid

Используется для создания сеточных макетов. Grid позволяет точно задавать размеры и размещение элементов на странице.

  • Гибкость при создании сеток
  • Определение четких областей для элементов
  • Поддержка адаптивных макетов
